A Leitura do dia mostra só produtos que também estão na calculadora TikTok (com ID BlueFocus). Use a conexão e a lista monitorada nas seções abaixo para atualizar.

Leitura do dia

Lista apenas produtos da calculadora TikTok com ID BlueFocus ou SKU só numérico (mesmo ProdutoId do PDV), e linha correspondente na lista monitorada. Com a página aberta, a leitura roda também automaticamente todo dia às 16h (horário do seu computador); os botões continuam disponíveis.

Conexão (SOAP)

O PDF da BlueFocus envia o token no header autentica. Use os mesmos códigos do login do PDV — em geral o EmpresaId é um código curto (ex.: SANHERO), não a razão social completa com espaços. Se a API recusar, confira no BlueFocus o código exato da empresa e do PDV. Salvar grava no JSON central (data/touya-db.json) via servidor.py. Você pode manter um modelo sempre pronto em db/bluefocus-soap-config.json (mesmos campos): se a config no banco estiver vazia, o servidor e a página completam a partir desse arquivo. O terminal do servidor só registra chamadas reais — use Testar conexão, Atualizar leitura ou Importar cadastro. Abra em http://localhost:8080 (não arquivo direto).

Importar cadastro (SOAP — ExportaCad SAT)

Operação IntegracaoFcxExportaCadSAT.Execute: corpo com Sdtwebserviceentradaexpcadastro (EmpresaId, UsuarioId, PDVCodigo, TipoAtualizacao, Tipo=4, ProdutoId para paginação, demais zerados). O proxy envia o header SOAPAction: "Valimaction/AINTEGRACAOFCXEXPORTACADSAT.Execute" e autentica, como no WSDL.
